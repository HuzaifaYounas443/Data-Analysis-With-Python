# COVID-19 Global Data Analysis

A comprehensive data analysis project exploring global COVID-19 statistics including confirmed cases, deaths, recoveries, and regional trends. This project provides deep insights through statistical analysis, visualizations, and interactive dashboards.

## Project Overview

This project performs an in-depth analysis of COVID-19 data across countries and WHO regions. It includes data cleaning, exploratory data analysis (EDA), statistical analysis, and creates various visualizations to understand the pandemic's global impact.

## Features

- **Data Cleaning & Preprocessing**: Handle missing values, convert data types, and calculate derived metrics
- **Global Statistics**: Total confirmed cases, deaths, recoveries, and active cases worldwide
- **Top Country Rankings**: Identify countries with highest cases, deaths, and recovery rates
- **Regional Analysis**: Compare statistics across WHO regions (Africa, Americas, Eastern Mediterranean, Europe, South-East Asia, Western Pacific)
- **Mortality & Recovery Analysis**: Calculate and compare mortality and recovery rates globally
- **Weekly Trends**: Analyze week-over-week changes and growth patterns
- **Outlier Detection**: Identify statistical outliers using Z-score methodology
- **Correlation Analysis**: Study relationships between different metrics
- **Risk Assessment**: Create composite risk scores based on multiple factors
- **Interactive Visualizations**: Generate both static (matplotlib/seaborn) and interactive (plotly) charts
